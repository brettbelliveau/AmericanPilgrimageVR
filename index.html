Project Description & Motivation

Our scene is a VR experience taking place during the Westward expansion of the 1800's. The user is given a glimpse of life on the Oregon Trail, riding by wagon through a desert environment. The scene begins with a couple minutes of dialogue, followed by the death of one the passengers.

The motivation from this project stems from the general perception of 1800's Westward Expansion. Most teachings of the major migrations such as the Oregon Trail tend to focus on the achievement and accomplishments of the travelers rather than the hardship they put themselves through to explore new regions of the continent. Elements of Presence

Many elements in this project were designed to increase the user's sense of presence. These major elements included:

  • First-Person Character View: Allowing the user to see his/her own body rather than being a floating camera

  • Dialogue: The time-specific dialogue gives the user some historical insight and introduces the setting

  • Ambient Sounds: The combination of background noises (wind, horses, wagons)

Environment Design

The scene's environment was inspired by many concepts described in the VR Book. The most significant of these are as follows:

  • Background: The terrain extends in all directions to each horizon until it meets the night sky. This scenery exists at every point in the scene.

  • Fundamental Geometry: Complementing the distant terrain, the scene's path has been scattered with trees, bushes and smaller hills. All of these have natural physics and colliders to add a closer sense of presence.

  • Interactive Objects: Throughout the path there are also movable pieces of terrain. They can only be interacted with by the main user (not other NPCs) and are central to the story.

  • Color and Lighting: The lighting implemented is bright enough to allow the user to see all the models in the scene, yet it is dark enough to maintain the feel of a night sky.

  • Audio: As mentioned earlier, the ambient noise and dialogue keep the user fully involved with the environment.

  • Environmental Wayfinding: The entire scene takes place on a one-way path and moves in one direction. This simply eliminates the need for any artificial wayfinders or visual aids.

Assets & Models

Here is a list physical of elements we used to build the scene (with the package and developer if imported):

  • Terrain (Original): Built using the Unity Terrain Editor
  • Sand Texture: "Free Sand Materials" - Nobiax/Yughues
  • Trees & Bushes: "Free Desert Plants" - Lemuria
  • Human Models (Original): Built using the Autodesk Character Generator
  • Human Animations (Original): Built using Mixamo
  • Wagon (Original): Built using Maya
  • Horses: http://www.orangefreesounds.com/walking-horse-carriage/
  • Crates & Miscellaneous Materials: "Pro Western Starting Pack" - Tamarin Studios
  • Night Sky: "Sky5X One" - RKD
